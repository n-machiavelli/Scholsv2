Scholarship Purpose:

Department: Financial Aid Office - Provost.

College: Financial Aid Office - Provost.

Requirements

Minimum Undergraduate GPA: 2.75

High School GPA: 2.75

Essay: An Essay is required towards applying for this scholarship.

Financial Need: This scholarship requires a student to have a Financial Need to be eligible to receive the scholarship. In order to establish "Financial Need", a student must file the Free Application for Federal Student Aid (FAFSA). You can file the FAFSA at www.fafsa.ed.gov.

This scholarship is not open to International Students.

There is a community service requirement in applying for this scholarship.

Leadership: There is a leadership experience requirement in applying for this scholarship.

There is no reference letter required for this scholarship
